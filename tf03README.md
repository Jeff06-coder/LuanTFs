# Final project request

A proposta do trabalho final desta aula é desenvolver um exercício prático em JavaScript com foco nos conceitos centrais estudados sobre tipos de referência.

O trabalho deve se concentrar na criação de um objeto principal mais completo, contendo propriedades simples e também um array interno. A partir desse objeto, o aluno deverá demonstrar no código como acessar, percorrer, comparar, copiar e transformar dados.

O foco da atividade não está em criar uma narrativa, mas sim em organizar bem os conceitos da aula dentro de um único arquivo executável no Node.js.

🎯 Objetivo Geral

Desenvolver um exercício em JavaScript utilizando um objeto principal com array interno, aplicando os conceitos centrais da Aula 03 em um único arquivo chamado tf03.js.

🧩 O que deve ser desenvolvido

O aluno deverá criar um repositório para o trabalho e, dentro dele, entregar um arquivo único chamado tf03.js.

Nesse arquivo, o aluno deverá criar um objeto principal que contenha:
- pelo menos 3 propriedades simples
- pelo menos 1 array dentro dele

A partir desse objeto, o código deve demonstrar de forma prática:

- criação e uso de object
- uso de array dentro de object
- iteração nas propriedades do objeto com for...in
- iteração nos itens do array com for...of
- comparação entre tipos de referência
- exemplo de cópia por referência
- exemplo de cópia com spread operator
- transformação do objeto em JSON com JSON.stringify()

O código deve ser escrito de forma organizada e legível, com saídas no terminal mostrando cada etapa do exercício.

Ao final, o aluno deverá entregar o link do repositório contendo o trabalho, com o arquivo tf03.js disponível para avaliação.
